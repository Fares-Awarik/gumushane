## About
A website built for a honey producer selling natural 
highland honey from the Gümüşhane region. The site 
presents the product range and the producer's story, 
and turns visitors into direct order inquiries.

## Features
- Product showcase with dynamic content (Laravel + MySQL)
- Producer story & trust-building sections
- Direct order / contact flow
- Responsive Turkish-language UI

## Tech Stack
Laravel · Blade · MySQL · Vite

## Client
Built for a real honey producer in Mersin, Turkey 🍯
